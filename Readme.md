# covid19monitoring_other_Weather

## Grundlage 
Dieses Skript bringt die täglichen Wetterparameter ([Quelle: meteoschweiz](https://data.geo.admin.ch/ch.meteoschweiz.klima/nbcn-tageswerte/nbcn-daily_SMA_current.csv)) der Station SMA Zürich Fluntern in das für die Integration in den [harmonisierten Datensatz "Gesellschaftsmonitoring COVID19"](https://raw.githubusercontent.com/statistikZH/covid19monitoring/master/covid19socialmonitoring.csv) benötigte Format.  Einige Wetterparameter wie etwa die Schneehöhe werden weggelassen.
